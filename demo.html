✅ 功能已成功集成！ 您的作品集现在支持Google Docs Viewer和嵌入式视频播放器。

📄 Google Docs Viewer 演示

以下是使用Google Docs Viewer展示PDF文档的示例：

// 在 assets/js/main.js 中配置： { id: 1, title: "示例文档", description: "这是一个示例PDF文档", filePath: "assets/document.pdf", googleDriveId: null, fullUrl: "https://yourusername.github.io/repository/assets/document.pdf", coverImage: null }
💡 提示： 这是一个示例PDF文档。在实际使用中，请替换为您自己的文档URL。

🎥 YouTube 视频嵌入演示

以下是嵌入YouTube视频的示例：

// 在 assets/js/main.js 中配置： { id: 1, title: "演示视频", description: "YouTube视频嵌入示例", filePath: null, duration: "1:30", youtubeId: "dQw4w9WgXcQ", thumbnail: "assets/images/video-thumb.jpg" }

⚙️ 配置说明

📋 文档配置选项：

  • filePath: 本地PDF文件路径
  • googleDriveId: Google Drive文件ID（推荐）
  • fullUrl: 完整的文件URL（用于GitHub Pages）

🎬 视频配置选项：

  • filePath: 本地视频文件路径（MP4格式）
  • youtubeId: YouTube视频ID
  • thumbnail: 视频缩略图路径
⚠️ 注意事项：
  • Google Drive文件需要设置为"任何人都可以查看"
  • YouTube视频需要允许嵌入
  • 本地文件需要确保路径正确
  • 建议PDF文件不超过50MB

🔧 如何使用

步骤1：配置文档数据

// 编辑 assets/js/main.js 文件 const documentsData = [ { id: 1, title: "您的文档标题", description: "文档描述", filePath: "assets/your-document.pdf", googleDriveId: null, // 或者填入Google Drive文件ID fullUrl: null, coverImage: null } ];

步骤2：配置视频数据

// 编辑 assets/js/main.js 文件 const videosData = [ { id: 1, title: "您的视频标题", description: "视频描述", filePath: "assets/your-video.mp4", // 本地视频 duration: "2:33", youtubeId: null, // 或者填入YouTube视频ID thumbnail: null } ];

步骤3：上传文件并测试

  1. 将PDF和视频文件上传到 assets/ 目录
  2. 或者上传到Google Drive并获取文件ID
  3. 更新配置文件
  4. 在浏览器中测试功能
